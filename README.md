# SlayerRewards Extension

**Path:** `plugins/MCEconomy/extensions/SlayerRewards`

SlayerRewards is a dynamic extension for **MCEconomy** that allows server administrators to fully customize rewards for killing mobs, including monetary payouts, custom item drops, and visual feedback.

* **GUI Editor:** Manage loot tables and drop chances entirely in-game.

* **Complex Rewards:** Support for random money ranges (e.g., 10-50 coins) and custom NBT items.
* **Vanilla Control:** Toggle default mob drops (like Rotten Flesh) on or off per mob.
